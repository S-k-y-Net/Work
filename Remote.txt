Настройка удаленного подключения к рабочему месту с помощью вашего ПК\ноутбука[править]
Для подключения к своему рабочему месту из дома или из любого другого места, необходимы следующие условия:

Наличие подключения к сети Интернет
Установленный на компьютере RDP client версии 6.0 или выше[1]
W-MSC-018  192.168.200.117
Для настройки удаленно подключения выполните следующие действия:

Запустите Подключение к Удаленному рабочему столу (Start\All Programs\Accessories\Remote Desktop Connection)
В появившемся окне введите имя своего компьютера или IP-адрес, нажмите кнопку Options
В появившемся окне выберите закладку Advanced, нажмите кнопку Settings…
В появившемся окне выберите Use these RD Gateway server settings:, введите tsg.terralink.ru, выберите Use my RD Gateway credentials for the remote computer, нажмите ОК, затем Connect
